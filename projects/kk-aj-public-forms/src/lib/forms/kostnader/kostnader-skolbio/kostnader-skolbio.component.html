Kostnader
Ange kostnad/pris för arrangemanget i SEK. Endast siffror. För paketpris använd fältet ”Övrigt om kostnader”. Här kan du också ange ev entréavgifter, kostnad för guidning, museilektioner etc (ingår inte i arrangörsstödet).
Vänligen fyll i kostnad/pris för arrangemanget i SEK. Endast siffror.
Här kan du ange om det finns rabatter, paketpris eller om det tillkommer fler kostnader, t.ex. entréavgifter eller liknande.
Övrigt
Fyll i om det är något mer som arrangören behöver veta om/inför arrangemanget.
